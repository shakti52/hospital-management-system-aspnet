Hospital Management System (ASP.NET)
Overview

The Hospital Management System is a web-based application built using ASP.NET Core MVC that helps manage hospital operations such as patient records, doctor details, appointments, and prescriptions in an organized and efficient manner.

This project demonstrates full-stack development using C#, Entity Framework Core, and SQL-based data handling.

Features

Doctor Management

Add, edit, view, and delete doctor details

Patient Management

Maintain patient records and medical history

Appointment Scheduling

Book and manage appointments between doctors and patients

Prescription Management

Create and track prescriptions for patients

CRUD Operations

Full Create, Read, Update, Delete functionality for all modules
Tech Stack

Frontend: HTML, CSS, Bootstrap
Backend: ASP.NET Core MVC (C#)
Database: SQL Server / Entity Framework Core
IDE: Visual Studio

Project Structure

hospital_management/
